# Exploratory Data Analysis for Netflix Data

This is a personal learning project made with  
https://www.kaggle.com/datasets/shivamb/netflix-shows/data  
(data added to Kaggle by Shivam Bansal)  



## Goals

* Discover and understand main features of this data set
* Understand the structure of the data 
* Do basic data preprocessing
* Find patterns and trends in the data
* Visualize the cleaned data  


##### Python Libraries Used

* Pandas
* NumPy  
* Matplotlib
* Seaborn
